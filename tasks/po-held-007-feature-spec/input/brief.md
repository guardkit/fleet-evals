# Feature brief: kiln firing slot booking

Studio members need to book firing slots in our two kilns. A slot covers one
firing. Members should be able to see when slots are free, book one, and
cancel a booking they no longer need. The studio manager needs to be able to
block out slots so a kiln can be taken out of use for servicing.

That is the whole ask from the studio — they have not told us anything else
yet.
